
# 🏥 MedJargon NLP Simplifier

An NLP-powered engine that detects complex medical terminology and suggests simpler alternatives using Lemmatization.

## 👥 The Team Workflow
* **Phase 1: EDA (Person 1)** - Data cleaning and statistical analysis of medical datasets.
* **Phase 2: NLP Detection (Person 2)** - Built the detection brain using `spaCy` and custom entity mapping.
* **Phase 3: Evaluation (Person 3)** - Performance metrics and accuracy validation (90%+ success rate).

## 🚀 How to Run
1. Clone the repo.
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`

## 🛠 Tech Stack
* **Language:** Python
* **NLP Library:** SpaCy (en_core_web_sm)
* **Web Framework:** Streamlit
